About PokéAttractor





  • Get more than 293,000 historic Pokémon sighting data from Kaggle
  • Web scrapped more than 275,000 Pokémon tweets from Twitter
  • Data were collected from July, 2016 to January, 2017
  • Stored in SQL database
  • Data cleaning and exploration in Python Pandas
  • kNN modeling and k-means clustering in Scikit-Learn
  • Natural language processing in NLTK
  • Time series analysis in Scipy
  • Geocoding in Google map API
  • Exploratory plots in Matplotlib, Seaborn
  • Interactive plots in Javascript D3, Folium, Bokeh
  • Web app with Bootstrap, CSS, Flask and Heroku
